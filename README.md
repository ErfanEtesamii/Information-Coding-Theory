# پروژه شبیه‌سازی سیستم مخابراتی (هافمن + همینگ + BSC)

درس نظریه اطلاع و کدگذاری - دکتر سبحانی

## توضیح پروژه
پیاده‌سازی زنجیره کامل یک سیستم انتقال اطلاعات دیجیتال:
1. تعریف منبع و تولید دنباله تصادفی
2. کدگذاری منبع با هافمن (و توسعه منبع)
3. کدگذاری کانال با کد همینگ
4. شبیه‌سازی کانال BSC و کدگشایی بر اساس سندروم
5. ارزیابی نرخ خطا (Pe) و مقایسه با حد شانون

## ساختار پروژه
src/
  source.py    -> فاز ۱: منبع و تولید دنباله
  huffman.py   -> فاز ۲: کدگذاری هافمن
  hamming.py   -> فاز ۳: ماتریس‌های H و G کد همینگ
  channel.py   -> فاز ۴: شبیه‌سازی BSC و کدگشایی
  evaluate.py  -> فاز ۵: محاسبه Pe و رسم نمودار
  main.py      -> منوی اجرای برنامه

## اجرا
pip install -r requirements.txt
python src/main.py

## وضعیت فعلی
