# پروژه شبیه‌سازی سیستم مخابراتی (هافمن + همینگ + BSC)

درس نظریه اطلاع و کدگذاری - دکتر سبحانی

## توضیح پروژه

پیاده‌سازی زنجیره کامل یک سیستم انتقال اطلاعات دیجیتال:

1. تعریف منبع و تولید دنباله تصادفی
2. کدگذاری منبع با هافمن (و توسعه منبع)
3. کدگذاری کانال با کد همینگ
4. شبیه‌سازی کانال BSC و کدگشایی بر اساس سندروم
5. ارزیابی نرخ خطا (Pe) و مقایسه با حد شانون

## ساختار پروژه

- `src/source.py` → فاز ۱: تعریف منبع و تولید دنباله
- `src/huffman.py` → فاز ۲: کدگذاری هافمن
- `src/hamming.py` → فاز ۳: ماتریس‌های H و G کد همینگ
- `src/channel.py` → فاز ۴: شبیه‌سازی کانال BSC و کدگشایی
- `src/evaluate.py` → فاز ۵: محاسبه Pe و رسم نمودار
- `src/main.py` → منوی اجرای برنامه

## اجرا

نصب پیش‌نیازها:
```bash
pip install -r requirements.txt
```

اجرای برنامه:
```bash
python src/main.py
```
